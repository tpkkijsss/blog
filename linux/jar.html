    # 最全的打包命令

    # 1. 目前工作接触到java文件打包相关的文件有两种:[.jar文件和.war文件]

    # 1.1 .jar文件打包命令

    解压命令：jar xvf  Example.jar (一般格式：jar xvf Filename.jar)

打包命令：jar cf  Example.jar  /path/filename1/*

    # 1.2 .war文件打包命令

    解压命令: jar xvf  Example.war (一般格式：jar xvf Filename.war)

打包命令: jar cf  Example.war  /path/filename1/*

    # 2. 其它Linux常用打包命令整理

    # 2.1 .tar格式

    解压命令: tar xvf 文件名.tar

打包命令: tar cvf 文件名.tar 目录

    # 2.2 .tar.gz格式

    解压命令: tar zxvf 文件名.tar.gz

打包命令: tar zcvf 文件名.tar.gz 目录

    # 2.3 .tar.bz2格式

    解压命令: tar jxvf 文件名.tar.bz2

打包命令: tar jcvf 文件名.tar.bz2 目录

    # 2.4 .tar.bz2格式

    解压命令: tar jxvf 文件名.tar.bz2

打包命令: tar jcvf 文件名.tar.bz2 目录

    # 2.5 .tar.Z格式

    解压命令: tar Zxvf 文件名.tar.Z

打包命令: tar Zcvf 文件名.tar.Z 目录

    # 2.6 .tar.tgz格式

    解压命令: tar zxvf 文件名.tar.tgz

打包命令: tar zcvf 文件名.tar.tgz 文件名

    # 2.7 .gz格式

    解压命令: gunzip 文件名.gz

解压命令: gzip -d 文件名.gz

打包命令: gzip 文件名

    # 2.8 .bz格式

    解压命令: bzip2 -d 文件名.bz

打包命令: bunzip2 文件名.bz

    # 2.9 .bz2格式

    解压命令: bzip2 -d 文件名.bz2

解压命令: bunzip2 文件名.bz2

打包命令: bzip2 -z 文件名

    # 2.10 .Z格式

    解压命令: uncompress 文件名.Z

打包命令: compress 文件名

    # 2.11 .zip格式

    解压命令: unzip 文件名.zip

打包命令: zip 文件名.zip 目录

    # 2.12 .lha格式

    解压命令: lha -e 文件名.lha

打包命令: lha -a 文件名.lha 文件名

    # 2.13 .rar格式

    解压命令: rar a 文件名.rar

打包命令: rar e 文件名.rar
